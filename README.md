
# Laravel Mailchimp OAuth Integration Boilerplate

This boilerplate provides a solution for integrating Mailchimp OAuth authentication into your Laravel application. It includes setup for connecting to Mailchimp's API, styled with Tailwind CSS.

## Installation

### Clone the Repository

```
git clone https://github.com/frahjokhio/laravel-mailchimp-oauth.git
cd laravel-mailchimp-oauth
```

### Install Dependencies

```
composer install
```

### Configure Environment Variables

Update your .env file with your Mailchimp APP credentials:

```
MAILCHIMP_CLIENT_ID={your_mailchimp_client_id}
MAILCHIMP_CLIENT_SECRET={your_mailchimp_client_secret}
```

### Serve the Application

```
php artisan serve
```

### Access the Page

Navigate to http://localhost:8000.
